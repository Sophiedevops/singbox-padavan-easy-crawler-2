Продвинутый, максимально оптимизированный скрипт автоматического поиска, парсинга, тестирования и ротации бесплатных прокси (Shadowsocks, VLESS, VMess, Trojan, Hysteria2) для ядра **Sing-Box** на роутерах с прошивкой **Padavan**.

Скрипт решает главную проблему слабых роутеров (MIPSLE/MIPS): он способен "переваривать" гигантские базы на 60 000+ прокси-ссылок, находить среди них лучшие серверы с шифрованием и применять их **без падения системы от нехватки оперативной памяти (OOM) и перегрева процессора**.

---

## ✨ Ключевые особенности (Почему этот скрипт особенный?)

* 🛡️ **Anti-OOM Uniform Sampling:** Защита от нехватки памяти. Скрипт не пытается загрузить все 60 000+ нод в ОЗУ. Он использует математическое равномерное сэмплирование (через `awk`), вытягивая 4500 репрезентативных ссылок со всей базы для безопасной обработки конвертером.
* 🧠 **Глобальная Матрица Приоритетов (Smart Sorting):** Вы сами решаете, что важнее:
  * `Протокол -> Страна`: Сначала собрать легкие для роутера зашифрованные Shadowsocks со всего мира, чтобы спасти CPU от 100% нагрузки.
  * `Страна -> Протокол`: Сначала выжать все доступные протоколы из конкретной страны (например, NL), чтобы получить минимальный пинг для игр.
* 🔒 **Параноидальный Режим Шифрования:** Умеет жестко отсекать "голые" и небезопасные протоколы (VLESS без TLS, Shadowsocks `method: none`, устаревшие уязвимые шифры вроде `rc4-md5`), оставляя только современный AEAD.
* 🧩 **Авто-Декодер Base64 & Санитайзер:** Встроенный очиститель мусора. Успешно переваривает битые подписки, удаляет нулевые байты (`\0`) и символы Windows (`\r`), которые ломают встроенный `grep` на Padavan.
* ⚡ **Smart Fast-Check (Умная проверка пула):** Скрипт не сканирует базу впустую. Сначала он проверяет текущие ноды в конфиге. Если живы 70% серверов, он просто отменяет полное сканирование, сберегая ресурс USB-флешки и процессора.
* 🔄 **Zero-Downtime:** Применение нового конфигурационного файла и перезапуск `sing-box` происходят на лету, не обрывая ваши текущие скачивания.
* 📦 **Параллельное Тестирование:** Скрипт тестирует ноды микро-батчами по 3 штуки, с идеально выверенными таймаутами для "холодного старта" MIPS-архитектуры.

## ⚙️ Требования (Requirements)

Для работы скрипта ваш роутер должен соответствовать следующим требованиям:
1. Прошивка **Padavan** (или аналогичная на базе Linux).
2. Подключенный USB-накопитель с установленной средой **Entware** (папка `/opt/`).
3. Установленные пакеты: `opkg install curl jq lua openssl-util bash coreutils-sort wget`.
4. Исполняемый бинарник ядра `sing-box` (сборка `linux-mipsle-softfloat` или `hardfloat` в зависимости от чипа) + настроенный базовый конфиг `conf3_final.json`.
5. Lua-конвертер (файлы `converter.lua` и `utils.lua`) в рабочей папке.
